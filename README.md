# 🚀 JavaScript Projects Portfolio

Welcome to my JavaScript Projects repository! This collection of JavaScript mini-projects was created as part of the **Dev Weekends Fellowship**.

These projects focus on building a strong foundation in DOM manipulation, event handling, and core JavaScript logic without the use of any external libraries or frameworks.

## 🛠️ Technologies Used

* **HTML5** - Page structure and forms

* **CSS3** - Styling and responsive layouts

* **JavaScript (ES6+)** - Interactive logic and DOM manipulation

## 📂 The Projects

### 1. Background Color Changer

An interactive utility that dynamically updates the webpage's background and text colors based on user selection.

* **Concepts Learned:** `querySelectorAll`, `forEach` loops, Event Listeners (`click`), and modifying inline CSS properties via the DOM (`element.style`).

### 2. BMI Calculator

A health tool that takes a user's height and weight, calculates their Body Mass Index (BMI), and provides an instant health categorization (Underweight, Normal, Overweight).

* **Concepts Learned:** Form handling, preventing default form submission (`e.preventDefault()`), input parsing and validation (`isNaN`), and injecting dynamic HTML (`innerHTML`).

* **Preview:**
  

### 3. Local Time Clock

A digital clock that continuously fetches and displays the user's current system time, updating accurately every second.

* **Concepts Learned:** The JavaScript `Date` object, `toLocaleTimeString()`, and asynchronous timing events using `setInterval()`.

* **Preview:**
  

### 4. Number Guessing Game

An interactive game where the player has 10 attempts to guess a randomly generated number between 1 and 100. It includes visual feedback (Too High/Too Low), attempt tracking, and a dynamic "Start New Game" feature.

* **Concepts Learned:** Number generation (`Math.random`, `Math.floor`), advanced conditional logic (win/loss states), disabling/enabling input fields, and dynamically creating/appending new DOM elements (`createElement`, `appendChild`).

## 💻 How to Run Locally

Since these projects are built with standard web technologies, there is no complex setup, installation, or local server required!

1. Clone this repository to your local machine
2. Navigate into the project folder you want to view.
3. Simply double-click the `.html` file (e.g., `bgColor.html`, `bmi_generator.html`) to open it in your default web browser.
